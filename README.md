# Fixed effects model

This python code replicates the main results in [*Jason Abrevaya. 2006. "Estimating the effect of smoking on birth outcomes using a matched panel data approach." Journal of Applied Econometrics*](https://onlinelibrary.wiley.com/doi/abs/10.1002/jae.851).

The paper's data set is available [here](http://qed.econ.queensu.ca/jae/2006-v21.4/abrevaya/).
